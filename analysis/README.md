# 📈 Exploratory Data Analysis & Statistical Modeling

This directory contains the data science and analytical components of the Vancouver Fuel Price Optimization project. 

The primary objective of this notebook is to statistically validate local heuristics regarding fuel pricing—specifically, the hypothesis that fuel prices in Richmond experience significant temporal drops during late-night hours compared to neighboring municipalities.



## 🎯 Analytical Methodology

The Jupyter Notebook (`Vancouver_Gas_Price_Analysis.ipynb`) follows a structured data science workflow:
1. **Data Wrangling & Cleansing:** Ingesting raw CSV data generated by the Google Apps Script pipeline, handling missing values, and parsing timestamps for time-series analysis.
2. **Exploratory Data Analysis (EDA):** Visualizing price distributions, daily moving averages, and geographic variances using `matplotlib` and `seaborn`.
3. **Statistical Inference (ANOVA):** Applying Analysis of Variance to determine if the price differences between specific time windows (e.g., 11 PM vs. 8 AM) and locations (e.g., Richmond vs. Vancouver) are statistically significant, rather than random noise.

## 🗂️ Directory Contents

* `Vancouver_Gas_Price_Analysis.ipynb` - The core Jupyter Notebook containing all Python code, statistical models, and visualizations.
* `requirements.txt` - A strict list of Python dependencies required to run the analysis, ensuring environment reproducibility.
* `sample_data.csv` *(Optional)* - A static snapshot of the fuel price data used to render the notebook, allowing users to test the code without connecting to the live database.
